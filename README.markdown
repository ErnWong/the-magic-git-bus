# The Magic Git Bus

This is an interactive guide to some of git's internal data structures and plumbing commands that runs in the browser.

![Loading screen with ducks in a school bus floating on water waiting for a deep dive into the depths of git](./screenshot-loading-screen.png)

![Screen of the webpage with some terse guide on the left, a working v86 terminal on the right to try out the git commands, and a git graph visualizer at the top that reflects the state in the emulator.](./screenshot.png)
